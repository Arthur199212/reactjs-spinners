## Get started

Install the package with `npm` or `yarn`

```sh
npm i react-css-spinners

yarn add react-css-spinners
```

Import spinner you like and use it

```ts
import { Ellipsis } from 'reactjs-spinners'

const AppLoader = props => (
  <Ellipsis />
)
```
